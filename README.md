<h1>PokemonFinder</h1>
<p>PokemonFinder, this app lets you search your favourite pokemon and fetch it's data.
Currently over 900 pokemon avaliable.</p>
<p>To run this app locally just install few dependendencies written in package.json</p>
<ul>start app scripts:
  <li>npm run start - runs app locally.</li>
  <li>npm run startDev - if you install nodemon in dev dependencies runs and watches any changes on app localy.</li>
  <li>npm run renderPug - renders pug and watches any changes in views directory.</li>
</ul>
